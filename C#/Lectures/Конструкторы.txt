Итоги

1. Есть типы возвращаемых значений, которые вмонтированы в процессор. Самыми распространнеыми из них есть

int
double
string
char
float
bool

2. Из типов данных которые вмонтированы в процессоре
int
double
string
char
float
bool
можно создавать собвственные типы данных с помошью ключевого слова class



3. Слово class: означает что создается описание транзисторов, которые могут быть задействованы с помощью загрузки их в процессор

4. Слово static: позволяет создавать методы свойства и поля, которые будут загружены в процессор на стадии запуска программы. Недостаток
таких методов, свойств и полей в том что они не могут быть расширяемыми

5. Оператор new: загружает описанные транзисторы в процессор

6. Любое имя переменной/поля есть не чем иным как адресным пространство к которому можно обратиться. А также чере точку можно обратиться
 к функциям методам и полям
данной переменной.
